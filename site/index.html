Quickstart

Clone, build, and run in under a minute.

git clone https://github.com/howieduhzit/bubblefetch.git
cd bubblefetch
make run
bf --help
yay -S bubblefetch-git
bf -o ~/Pictures/bubblefetch.svg

Configuration

Copy the sample config, then edit modules and themes to taste.

cp config.example.yaml ~/.config/bubblefetch/config.yaml

Style-first themes

Try curated themes like dracula, nord, or tokyo-night and ship your own JSON themes in seconds.

Run themed

make run-dracula

Theme files

Drop JSON in ~/.config/bubblefetch/themes/.
