# Documentation Index

This folder contains the core documentation for bubblefetch. Start with the quickstart,
then dive into examples, plugins, and performance notes.

## Quick links

- `QUICKSTART.md` - 60-second setup and basic usage
- `EXAMPLES.md` - Real-world usage patterns and exports
- `PLUGINS.md` - Plugin API and authoring guide
- `PERFORMANCE.md` - Benchmarks, speed notes, and optimization highlights
- `CHANGELOG.md` - Release notes and history

The landing page includes a plugin store powered by `plugins/manifest.json`.

- `site/` - GitHub Pages landing page and assets
- `examples/exports/` - Sample HTML/SVG exports
